# Youbube (YouTube CSV Dashboard)

## วิธีเปิดใช้งาน (สำคัญ)
1) แตกไฟล์โฟลเดอร์นี้
2) เปิดโฟลเดอร์ด้วย VS Code
3) ติดตั้งส่วนขยาย **Live Server**
4) คลิกขวา `index.html` → **Open with Live Server** (หรือกด Go Live)

> ห้ามดับเบิลคลิกเปิดไฟล์ผ่านเบราว์เซอร์ตรง ๆ เพราะเบราว์เซอร์จะบล็อกการอ่านไฟล์ CSV (CORS)

## เข้าสู่ระบบ
- Username: `admin`
- Password: `1234`
แก้ได้ที่ไฟล์ `js/auth.js`

## ไฟล์ข้อมูล
- `assets/data/youtube.csv` (ฝังไว้ในเว็บ ไม่ต้องอัปโหลด)

## โครงสร้างไฟล์
- `index.html` หน้า Login
- `app.html` หน้าเว็บหลัก (Sidebar + Tabs)
- `css/style.css` สไตล์สีสดใส
- `js/app.js` โหลด CSV, ทำ Filters, วาดกราฟ, ตารางแบ่งหน้า
- `js/auth.js` ระบบล็อกอินแบบง่าย

## ปรับแต่งได้
- เปลี่ยนชื่อ/เครดิตผู้จัดทำ: ใน `app.html` (แท็บ “ข้อมูลผู้จัดทำ”)
- เปลี่ยนสี: ใน `css/style.css` (ตัวแปร :root)


## ส่งออก PDF (รายงาน)
- กดปุ่ม **Export PDF (รายงาน)** มุมขวาบน
- ระบบจะส่งออก “หน้าที่กำลังเปิดอยู่” เป็น PDF ขนาด A4 แบบหลายหน้าอัตโนมัติ

## macOS (Mac) ถ้าข้อมูลไม่ขึ้น
บางครั้งเบราว์เซอร์บน Mac อาจบล็อกการอ่านไฟล์ CSV จากพาธภายใน (CORS)
- ให้เปิดผ่าน Live Server เป็นหลัก
- ถ้ายังไม่ขึ้น: กดปุ่ม **อัปโหลด CSV (สำรอง)** แล้วเลือกไฟล์ `youtube.csv` จากเครื่อง
  เว็บจะอ่านข้อมูลผ่าน FileReader โดยไม่ต้อง fetch
